### Contributing
Pull requests are accepted and greatly appreciated. Please be sure to follow the code guidelines stated bellow.

### Code Guidelines

- [SASS](https://sass-guidelin.es/)
- [HTML](https://www.w3schools.com/html/html5_syntax.asp)
